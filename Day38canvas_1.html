canvas

什么是canvas

  • canvas 元素用于在网页上绘制图形。HTML5 的 canvas 元素使用 JavaScript 在网页上绘制2D图像。
  • 在矩形区域的画布上，控制其每一像素，JavaScript 来绘制 2D图形，逐像素进行渲染。可以通过多种方法使用canvas 元素绘制路径、矩形、圆形、字符以及添加图像。
  • 图片处理,游戏,上传图片
  • canvas在页面上相当于一张图片
  • 注意点:style改变样式会拉伸或压扁图片
  • 浏览器支持情况:IE9.0以前的版本不支持
  • getContext() 方法可返回一个对象，该对象提供了用于在画布上绘图的方法和属性。

canvas的文字绘制

  • font设置或返回文本内容的当前字体属性
  • textAlign设置或返回文本内容的当前对齐方式--center left right
  • textBaseline设置或返回在绘制文本时使用的当前文本基线--middle top bottom Alphabetic(以文字一下面为基线可以超出) Hanging(以文字上方为基线)

  • fillText(text,x,y,maxWidth): text表示文字，x、y为坐标、maxWidth可选，为文字最大宽度，防止文字溢出；填充文字

  • strokeText(text,x,y,maxWidth): text表示文字，x、y为坐标、maxWidth可选，为文字最大宽度，防止文字溢出；绘制文字 对齐方式是左下角

阴影绘制

  • shadowColor 阴影颜色
  • shadowOffsetX 阴影的左右偏移值
  • shadowOffsetY 阴影的上下偏移值
  • shadowBlur 阴影模糊度 >需要注意的一点是封装好shadow函数,调用是需要放在文字书写之前
//阴影函数的封装
function shadow(x,y,blur,color){
        ctx.shadowOffsetX = x;
        ctx.shadowOffsetY = y;
        ctx.shadowBlur = blur;
        ctx.shadowColor = color;
    }
    //注意点:需要再ctx.strokeText()/ctx.fillText(),shadow要再这之前调用

rect封装矩形

function rect(x,y,w,h){
        ctx.beginPath();
        ctx.moveTo(x,y);
        ctx.lineTo(x+w,y);
        ctx.lineTo(x+w,y+h);
        ctx.lineTo(x,y+h);
        ctx.lineTo(x,y);
        ctx.fill();//一定要填充颜色
        ctx.closePath();
    }

贝塞尔曲线

公式:ctx.arc(x,y,r,0,Math.PI,false/true);x,y是曲线的圆心,r是半径,0 Math.PI是起始位置和结束位置, false/true是顺时针和逆时针

二次曲线

  • 封装贝塞尔曲线
var canvas = document.querySelector("canvas");
    var ctx = canvas.getContext("2d");
    canvas.onmousedown = function(e){
        var disX = canvas.offsetLeft;
        var disY = canvas.offsetTop;
        document.onmousemove = function(e){
            var x = e.clientX - disX;
            var y = e.clientY - disY;
            ctx.clearRect(0,0,400,400);
            draw(x,y);
        }
    }
    document.onmouseup = function(e){
        document.onmousemove = "";
    }
    draw(100,100);
    function draw(x,y){
        ctx.beginPath();
        ctx.moveTo(100,100);
        ctx.quadraticCurveTo(x,y,200,200);
        ctx.strokeStyle = "black";
        ctx.stroke();
        ctx.beginPath();
        ctx.moveTo(100,100);
        ctx.lineTo(x,y);
        ctx.strokeStyle = "red";
        ctx.stroke();
    }
